## Developing

This is tested on Fedora Workstation (GNOME Wayland)

### Prerequisites

These are required to build GLFW

```shell
sudo dnf install wayland-devel libxkbcommon-devel
```

### Build

Build the APP

```shell
make build
```

### Run

Build the APP and Run it

```shell
make run
```

### Build And Run

Same as `make run`. Just less keystrokes

```shell
make
```
